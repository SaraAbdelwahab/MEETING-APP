An extremely advanced, animated homepage for Vaultex - a secure video meeting platform.

### 🌍 Animated World Map
- Real-time animated connection arcs between global cities
- Pulsing city nodes with glow effects
- Dynamic data flow visualization
- SVG-based animations with smooth transitions
